# CivilVote Platform

**CivilVote Technologies, Inc.** — Political technology platform suite for campaigns, voters, and party operations.

## Platform Architecture

```
CivilVote (parent company)
├── keisha-governor/    — Keisha Lance Bottoms Campaign App (BlueVote-powered)
├── bluevote-app/       — BlueVote: Democratic campaigns platform ($125K/mo)
├── redvote-app/        — RedVote: Republican campaigns platform ($125K/mo)
├── swingvote-app/      — SwingVote: Independent voter intelligence ($140K/mo)
└── civicvote-hq/       — CivicVote HQ: Admin & operations dashboard (internal)
```

## Tech Stack

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Convex (real-time database + serverless functions)
- **Auth:** Viktor Spaces Slack OAuth
- **Hosting:** Vercel (production)
- **PWA:** Full progressive web app support (installable on mobile)
